Pesca 2020/2021

Este inquérito visa obter informação sobre o perfil dos praticantes de pesca, sobre a atividade exercida e, ainda, recolher dados previsto no âmbito da regulamentação europeia sobre determinadas espécies.

* A proteção das pessoas singulares no que diz respeito ao tratamento de dados pessoais pela AEA é regulada pelo Regulamento (CE) n.º 45/2001 do Parlamento Europeu e do Conselho, de 18 de dezembro de 2000.
